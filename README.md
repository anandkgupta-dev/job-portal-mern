# 💼 MERN Job Portal

A full-stack Job Portal / Freelance Marketplace built using MERN stack.

## 🚀 Features

- 🔐 JWT Authentication
- 👥 Role-Based Access (Recruiter / Job Seeker)
- 💼 Job Posting (Recruiter)
- 📄 Job Application System (Job Seeker)
- 📊 Application Tracking (Accept / Reject)
- 🔍 Job Filtering

## 🛠 Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT

## 📦 Installation

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
```
